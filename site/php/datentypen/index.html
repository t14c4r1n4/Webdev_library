Datentypen

string Text 
  $name = 'Max';
  echo $name; // => Max
  echo 'Hallo $name!'; // => Hallo $name!
  $name = "Max";
  echo "Hallo $name!"; // => Hallo Max!

in einfachen oder doppelten Anführungszeichen;

- "Escape-Sequenzen" (*/t* = Tabulator, */n* = Zeilenumbruch)

- "Auflösung" von Variablen in Werte

- Konkatenation

Composite Types

object Instanz einer Klasse kann aus mehreren Eigenschaften und Methoden bestehen
array Sammlung von Werten kann aus verschiedenen Datentypen bestehen und ist sehr flexibel
class Vorlage für Objekte kann als Vorlage für die Erstellung von Objekten verwendet werden; besteht aus Eigenschaften und Methoden und kann auch von anderen Klassen erben

Intersection Types (&)

ab PHP 8.0

Mithilfe vom &-Operator können mehrere Typen kombiniert werden. 

T&U => Typ T UND Typ U

Union Types (|)

ab PHP 8.0

Mit dem Pipe-(|)-Operator kann man eine ODER-Funktion kombinieren. 

T|U => Typ T ODER Typ U

Daten konvertieren

Casting

  $zahl = "1";
  var_dump($zahl);
  echo "<br />";
  $zahl = (int) $zahl;
  var_dump($zahl);

Output: 

  string(1) "1";
  int(1);

Type Juggling

PHP kann automatisch mit den Dateitypen "jonglieren". D.h. im Kontext konvertiert PHP den Dateityp:

numeric float, int
string echo, print, concatenation, string interpolation
logical <, =, >, AND, OR, XOR, !, &&, ||; => bool
integral and string &, |, ^(XOR), ~ (Nicht), <<, >>
comparative ==, ===, !=, <>, !==, <, >, <=, >=, <=>
function

wenn ein Rückgabewert deklariert wurde; der Wert muss dem deklarierten Wert des Typs entsprechen;

Ausnahmen:

  1. Wert ist vom Typ int und der deklarierte Typ ist float => int wird in float konvertiert;
  2. Wenn der deklarierte Wert ein scalar type ist, wird der Wert in einen scalar type konvertiert und der "coercive typing mode" (none-strict) ist aktiviert
